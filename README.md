# 🚚 Waze Delivery Automation

**Automação de Geração e Abertura de Rotas no Waze para Serviços de Entrega**

Robô desenvolvido em Python com **Playwright** para o Desafio Final do curso Python to Automation + RPA Intro.

## ✨ Funcionalidades

- Lê planilha Excel com as entregas
- Gera Deep Links oficiais do Waze (`https://waze.com/ul`)
- Abre cada rota automaticamente no navegador

## 🛠️ Tecnologias Utilizadas

- **Playwright** (automação web)
- Pandas + openpyxl (leitura Excel)
- Python 3.10+

## 🚀 Como rodar

1. Clone o repositório
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt

   ```
